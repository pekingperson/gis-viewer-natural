How to Use

Getting Started

  1. Load Map: Click "Load Map Image" to upload a JPEG or PNG map file.
  2. Calibrate: Use the "Calibrate" tool and set at least 2 reference points with known coordinates.
  3. Navigate: Use mouse wheel to zoom, drag to pan.

Tools

  • Pan: Default mode for navigation.
  • Calibrate: Set reference points for coordinate mapping.
  • Annotate: Add labeled points to the map.
  • Measure: Click two points to measure distance.

Features

  • Import GeoJSON files for overlays.
  • Import historical data from ImageMapper websites.
  • Toggle coordinate grid display.
  • Export annotations and data as JSON.
  • Persistent storage (data saved automatically).
